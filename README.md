# homebrew-tap

Homebrew formulae for [ChunzhengLab](https://github.com/ChunzhengLab) projects.

## Usage

```sh
brew tap ChunzhengLab/tap
brew install sure-rm
```

> **Note:** Formulae are published automatically when a new release is tagged.
> If a formula is not yet available, check the [releases page](https://github.com/ChunzhengLab/sure-rm/releases) for the latest version.

## Available Formulae

| Formula | Description |
|---------|-------------|
| [sure-rm](https://github.com/ChunzhengLab/sure-rm) | Safer rm replacement that moves files to a recoverable trash store |
